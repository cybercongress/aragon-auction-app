# Auction Aragon Dapp

## Description of flow

## Prepare
```
npm install @aragon/cli -g
```
Note: We recommend use node v10.18.0 and aragon CLI v7.0.4 or latest

## Development

In /app directory (app hot reload and script rebuild)
```
npm run start
```

```
npm run watch:script
```

In project root
```
npm run start:http:template
```
